SMChat

What is SMChat?

SMChat is a clojure coded graphical user interface (GUI) using the SeeSaw Library.

What features does SMChat have?

SMChat is still largely unfinished and being worked on.

  • Clearing Messages
  • Changing Prompt
  • Font Color
  • Theme
  • Login Feature

What lies in the future for SMChat?

  • Implementing chatbots.
  • A clear registration.
  • Access to SMChat through a web browser.
  • Creating a SMChat version for Linux.
  • A ton more.
